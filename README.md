# ⛅ 天氣猜謎遊戲

根據線索猜出天氣的小遊戲！10 題關卡，連勝加分，挑戰滿分！

## 🚀 本地執行

```bash
npm install
npm start
```

打開瀏覽器到 http://localhost:3000

## 📦 部署到 Railway

1. 把這個專案推到你的 GitHub repo
2. 前往 [railway.app](https://railway.app) 並登入
3. 點擊 **New Project** → **Deploy from GitHub repo**
4. 選擇這個 repo → 點擊 **Deploy Now**
5. Railway 會自動偵測 Node.js 並建置！
6. 建置完成後，點擊 **Generate Domain** 取得網址

就這樣！🎉
